# Patrick O'Mara Website – Astro & Tailwind CSS

This is my personal website built with [Astro.js](https://astro.build) and Tailwind CSS.  
It serves as a portfolio and blog, showcasing projects, writings, and insights into data, cloud engineering, and biotech.

![Patrick O'Mara Website](public/your-preview-image.jpg)

## Features

- ✅ Dark and light color mode
- ✅ Portfolio and blog sections
- ✅ Responsive, mobile-first layout
- ✅ Optimized images using Astro’s `Image` component
- ✅ SEO-friendly with canonical URLs and OpenGraph data
- ✅ Markdown & MDX support for blog posts
- ✅ Tailwind CSS for modern styling
- ✅ Clean, minimal design focused on content
